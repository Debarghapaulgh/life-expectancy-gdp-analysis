# GDP and Life Expectancy Analysis

## Overview

This project explores the relationship between GDP and life expectancy across six countries between 2000 and 2015 using Python-based data visualization techniques.

## Objectives

- Analyze trends in life expectancy over time
- Analyze GDP growth patterns across countries
- Investigate the relationship between GDP and life expectancy
- Support visual findings using correlation analysis

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The dataset contains GDP and life expectancy data for:

- Chile
- China
- Germany
- Mexico
- United States
- Zimbabwe

## Key Findings

- A strong positive relationship exists between GDP and life expectancy.
- The relationship exhibits diminishing returns at higher GDP levels.
- Countries with similar life expectancy can operate at vastly different economic scales.
- Zimbabwe demonstrates the significant impact of public health conditions on life expectancy.

## Visualizations

- Life Expectancy Trends (2000–2015)
- GDP Trends (2000–2015)
- GDP vs Life Expectancy (Log Scale)
- Correlation Analysis
